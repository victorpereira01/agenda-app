## 🏃 How to run?
```bash
# Clone this repo
$ git clone https://github.com/victorpereira01/sticky-notes-app

# Install the server dependencies
$ npm i 

# Start the server 
$ npm run dev

# Install the web dependencies
$ npm i

# Start the web
$ npm start
```

- Node.js
